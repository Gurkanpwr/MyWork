in order to use this code perfectly, Firstly, Nucleo-f446ZE board OR STM32f446ZETx chip is required. Secondly, project requires two USB connection, so another USB cable needs to connect to the board. To do this, a USB connector's "data -" cable needs to connect PB14 and "data +" needs to be connected to PB15 ports of the board. If this state isn't completed correctly, program won't respond to the user.
